# Azure DevOps MCP Proof of Concept (POC)

This repository contains two independent Model Context Protocol (MCP) application directories, hosted in their respective branches:

- **[`local-mcp`](https://github.com/Lavanya-2402/ADO-MCP-POC/tree/local-mcp)**: Contains the local MCP app code and configuration.
- **[`remote-mcp`](https://github.com/Lavanya-2402/ADO-MCP-POC/tree/remote-mcp)**: Contains the remote MCP app code and configuration (with OAuth / PKCE authentication tools).

## How to Switch Branches

To check out a specific application, switch to its branch:

```bash
# For local MCP app
git checkout local-mcp

# For remote MCP app
git checkout remote-mcp
```
